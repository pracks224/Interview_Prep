Ans 2. Autoboxing and Unboxing ,java compiler automatically converts the primitives to
       Wrapper object,hence using primitive inside the lambda function impacts performance.
       -> Hence to play around primitives use Functional interfaces etc.
        Ex: public interface IntSupplier (){
             int getAsInt()
            }
        Implementation ->
          IntSupplier sup = ()->18;
          int result  = sup.getAsInt();
        Similarly
        public interface IntPredicate(){
          boolean test(int n);
        }
        IntPredicate p = (age)->age>=18;
        boolean res = p.test(20);
